SOUND-STEREO-PAN

[GENERIC-FUNCTION]
OM 6.9 beta 0
Function Reference
ARGUMENTS:
  - S a sound  
  - LEFT a left channel pan value [default = -100]
  - RIGHT a right channel pan value [default = 100]

Description:

Pan a stereo sound.

<left> is a panoramic value for the left channel between -100 (full left) and 100 (full right).
<right> is a panoramic value for the right channel between -100 (full left) and 100 (full right).
