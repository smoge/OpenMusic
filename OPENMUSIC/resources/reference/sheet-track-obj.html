SHEET-TRACK-OBJ

[CLASS]
OM 6.9 beta 0
Function Reference
SLOTS:

- SELF :object
- OBJ :a musical object
- ID :object ID (integer)
- START-T :object onset (ms)

Description:

The SHEET-TRACK-OBJ is an internal class used to store objects in the sheet tracks. It is usually generated automatically but can be created in OM visual program as well.

<obj> is the musical object object.
<id> is used to identify the object in the sheet (for algorithmic manipulations).
<start-t> is the absolute onset of the object in the sheet track.
