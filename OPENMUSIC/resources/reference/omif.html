OMIF

[GENERIC-FUNCTION]
OM 6.9 beta 0
Function Reference
ARGUMENTS:
  - TEST IF  
  - ACTION THEN  
&optional
  
  - ELSE ELSE  

Description:

IF <test> THEN <action> ELSE <else>.

If the evaluation of <test> is not NIL, evaluates <action>.
Otherwise evaluates <else> (when supplied) or returns NIL.

Ex. (omif (= 4 4) 'A 'B) ==> 'A
Ex. (omif (= 4 5) 'A 'B) ==> 'B
Ex. (omif (= 4 5) 'A) ==> NIL
